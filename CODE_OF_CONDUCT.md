# Code of Conduct

This project adopts the [Contributor Covenant Code of Conduct, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — issues, pull requests, discussions, code, comments anywhere — you agree to abide by its terms.

## Reporting

To report a concern privately, contact the project maintainer at **dennis@ollhoff.life**. Reports are handled confidentially. Public issues are not the right channel — please reach out via email instead.

For the full text, FAQ, and translations, see <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
